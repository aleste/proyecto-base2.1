Bienvenido a Proyecto-base

Proyecto Base es una configuración inicial para el desarrollo de Aplicaciones y Sistemas de Gestión web utilizando Symfony 2. Incluye una selección de los mejores Bundles, Librerías y configuraciones que te permiten comenzar rápidamente un nuevo desarrollo.

¿Qué incluye el proyecto?

Una selección de los mejores Bundles y Librerías:

  • SonataAdminBundle
  • SonataDoctrineORMAdminBundle
  • FOSUserBundle
  • SimpleThingsEntityAuditBundle
  • RaulFraileLadybugBundle
  • LiuggioExcelBundle
  • WhiteOctoberTCPDFBundle
  • GregwarCaptchaBundle
  • Font Awesome
  • Twitter Bootstrap
  • jQuery
